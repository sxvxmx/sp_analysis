# Анализ данных пользователей

Проект для анализа event-логов пользовательской активности на платформе экскурсий.

## Требования

- Python 3.13+
- [uv](https://github.com/astral-sh/uv) (менеджер пакетов - опционально)

## Установка

1. Клонируйте репозиторий:
```bash
cd /path/to/lab/test
```

2. Установите зависимости через uv:
```bash
uv sync
```
3. или pip
```bash
pip install statsmodels, pandas, pyyaml, seaborn
```


## Запуск

### Ноутбук с анализом

Откройте основной ноутбук в Jupyter:
```bash
uv run jupyter test-task.ipynb
```

Или через VS Code:
```bash
code test-task.ipynb
```


## Структура проекта

```
test/
├── test-task.ipynb          # Основной ноутбук с анализом для первого задания
├── ab.ipynb          # Ноутбук с AB тестом для второго задания
├── datasets.yaml            # Конфигурация путей к данным
├── datasets/                # Исходные данные (CSV)
├── feature_*.py            # Модули с функциями анализа
├── city_to_country.py       # Словарь город→страна
└── README.md               # Этот файл
```

## Основные модули

| Файл | Описание |
|------|----------|
| `feature_filter_classification.py` | Классификация фильтров на 3 категории |
| `feature_normalize_filters.py` | Нормализация названий фильтров |
| `city_to_country.py` | Словарь для маппинга городов в страны |


## Результаты

После выполнения анализа вы получите:
- Статистику по использованию фильтров
- Распределение по городам и странам
- Топ популярных фильтров
- Анализ ценовых фичей
